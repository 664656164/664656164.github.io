爬虫的概念：

爬虫：（抓取网页的数据）1.使用程序模拟浏览器，去向服务器发送请求，获取响应信息。/2.通过一个程序，根据url进行爬取网页，获取有用信息。

反爬虫机制：门户网站，可以通过制定相应的策略或者技术手段，防止爬虫程序进行网站数据的爬取。

反反爬策略：爬虫程序可以通过制定相应的策略或者技术手段，破解门户网站中具备的反爬机制，从而可以获取门户网站中相关的数据。

beautifulsoup4（解析数据）

1.beautifulsoup4的概念

beautifulsoup库是一个可以解析HTML文件和XML文件的python库。

特点：

(1).beautifulsoup4提供了用于浏览，搜索和修改解析树的简洁函数，可以通过解析文档为用户提供需要抓取的数据。

(2).beautifulsoup4自动将输入文档稳定转化为Unicode编码，将输出文档转换为UTF-8编码，因此使用者不需要考虑编码方式。除非文档没有指定编码方式， 此时beautifulsoup4库不能自动识别编码方式， 需要说明原始编码方式。

(3) beautifulsoup4库能够为用户灵活地提供不同的解析策略或较快的交互速度。

beautifulsoup4库中的BeautifulSoup类非常重要, 一般通过导入此类来解析网页内容。导入此类的语法格式为：

from bs4 import BeautifulSoup

Tkinter图形用户界面

1. Tkinter概念

Tkinter（即 tk interface） 是 Python 标准 GUI 库，简称 “Tk”。Tkinter 编写的程序，也称为 GUI 程序，GUI （Graphical User Interface）指的是“图形用户界面”，在计算机中表示图形，以及利用计算机进行图形的计算、处理和显示等相关工作。

一个 GUI 程序一般由窗口、下拉菜单或者对话框等图形化组件构成， 通过鼠标点击菜单栏、按钮或者弹出对话框的形式来实现人机互动，从而提升人机交互的体验，让“冰冷”的程序变得有“温度”。

2. 基于Tkinter模块创建GUI程序的步骤

1.创建应用程序主窗口对象（跟窗口）

（1）通过类Tk的无参构造 root=Tk()

2.在主窗口中，添加各种可视化组件。比如按钮(Button)/文本框(Label)等

btn01=Button(root)

3.通过几何布局管理器，管理组件大小和位置

btn01.pack()

4.事件处理

通过绑定事件处理程序，响应用户操作所触发的事件。（比如单机或双击）

button1.bind(““,sendflower)

3. 主窗口位置和大小

root.geometry(“宽度x高度+/-x+/-y”) 进行设置

+x 表示距屏幕左边距离 -x 表示距屏幕右边距离

+y 表示距屏幕上边距离 -y 表示距屏幕下边距离

4.常用root方法

1.设置窗口名：root.title(“窗口名”)

2.设置窗口大小和位置：root.geometry(“宽度x高度+/-x+/-y”) ->宽度和高度是窗口大小，x,y是位置。

3.进入事件循环:root.mainloop()

5.GUI窗口面向对象写法

6.Lable标签

1.Lable标签主要用于显示文本信息,也可以显示图像信息。

2.Lable标签常用属性：

（1). width,height 指定文本区域大小，单位字符，一个汉字占俩个字符。如果是图片单位为像素。

(2).font 指定字体和字体大小 font=(字体,字号)

(3).image 显示在Label上的图像，目前tkinter只支持gif格式

(4).fg和bg

​ fg(foreground):前景色(字体颜色)，bg(background):背景色

(5). justify 针对多行文字的对齐，可设置 justify属性，参数 left/right/center

（6）borderwidth=? 边框的大小

（7）relief=”参数” 参数：solid（实线）/groove/ridge/flat/sunken/raised 边框线条

7.options选项

1.可以通过options设置组件的属性，从而控制组件的各种状态，比如，宽度，高度，颜色，位置等等

2.设置options选项三种方式：

​ (1).创建对象时，使用命名参数（也叫关键字参数）

​ fred = Button(self, fg=”red”, bg=”blue”)

（2).创建对象后，使用字典索引方式

​ fred[“fg”] = “red”

​ fred[“bg”] = “blue”

​ (3)创建对象后，使用config()方法

​ fred.config(fg=”red”,bg=”blue”)

8.Button

Button(按钮)用来执行用户的单击操作，Button可以包含文本，也可以包含图像，按钮被单击后会自动调用对应事件绑定的方法。

button = Button(root,text=”按钮名字”，command=绑定事件函数)

绑定一个事件的图片按钮

9.Entry单行文本框

Entry用来接受一行字符串控件。如果用户输入的文字长度长于Entry控件的宽度时，文字会自动向后滚动，如果想输入多行文本，需要使用Text控件。

entry.get() 获得文本框输入的内容

**Stringvar()**是tkinter里面的一个类型，用于获取文本框里面输入的内容

Doublevar(),Intvar(),Booleanvar()

用法：

Stringvar变量绑定到指定组件，Stringvar的值发生改变，组件内容文本框的值也发生改变；组件内容文本框的值发生改变，Stringvar的值也发生改变。（双向绑定）

n = Stringvar() ->定义一个Stringvar类型的变量

self.entry = Entry(self,textvariable=n)->textvariable属性 将变量n与文本框关联

注意：messagebox.showinfo(“星星蝶舞系统提醒您”, “欢迎登录星星蝶舞系统！！”) 消息弹框名：星星蝶舞系统提醒您 弹框内容为：欢迎登录星星蝶舞系统！！

*10.Text 多行文本框

[csdn详细教程](tkinter-Text详解_tkinter text-CSDN博客)

Text(多行文本框)主要用于显示多行文本，还可以显示网页链接，图片，HTML网页，甚至CSS样式表，添加组件等，因此，也常被当做简单的文本处理器，文本编辑器或者网页浏览器来使用。

  1. pack(side=”left”) 可以将组件放到左边位置

  2. self.text1.insert(行.列, “插入文本内容”)；在几行几列开始插入文本内容

    注意：text行从1开始，列从0开始

  3. self.text1.insert(INSERT，”插入文本内容”) INSERT索引表示在光标处插入文本内容

  4. self.text1.insert(END，”插入文本内容”) END索引表示在最后插入文本内容

    self.text1.insert(1.end，”插入文本内容”) 表示在第一行的最后插入文本内容

  5. self.text1.get(行1.列1,行2.列2); 返回行1.列1到行2.列2区间之间的文本内容

  6. self.text1.get(1.0,END); 返回所有文本内容

  7. self.text1.image_create(END,image=photo) 添加图片

  8. button = Button(self.text1,text=”按钮名字”)

​ self.text1.window_create(INSERT,window=button) 添加按钮组件

  1. self.text1.delete(1.0,END) 删除所有内容
